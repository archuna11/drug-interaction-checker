# 💊 Drug Interaction Checker

An AI-assisted clinical decision support tool that analyzes:
- Drug–Drug interactions
- Drug–Food interactions
- Drug–Disease interactions

---

## 🚀 Features

✅ Multi-drug interaction analysis  
✅ Severity-based grouping:
- 🚨 Critical
- ⚠️ Moderate
- 🟢 Minor  

✅ Dual mode interface:
- 👨‍⚕️ Doctor view (detailed)
- 👤 Patient view (simplified guidance)

✅ Prescription upload (OCR-based)

✅ Real-time UI with loading states

---

## 🧠 How it works

1. User enters drugs or uploads prescription  
2. Backend extracts drug names  
3. Pipeline checks:
   - Drug–Drug interactions
   - Food interactions
   - Disease interactions  
4. Results are grouped by severity  
5. Patient mode simplifies critical alerts  

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML / CSS / JavaScript
- OCR (for prescription parsing)
- Rule-based + AI-assisted pipeline
